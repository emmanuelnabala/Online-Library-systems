# Online Library System

## Description
This is a simple C++ Online Library System that demonstrates the use of
classes, header files, and object-oriented programming concepts.

## Files Used
- Book.h : Handles book details
- User.h : Handles user information
- Library.h : Manages library operations
- main.cpp : Program entry point

## Test Cases

### Positive Test Cases
- Adding a book with a valid ID and title
- Creating a user with correct details
- Displaying library information successfully

### Negative Test Cases
- Using invalid book ID (e.g., negative number)
- Creating a user with empty name
- Accessing book/user without initialization
